# personalized-task-manager
A simple task manager I built using Python and SQL while learning how to connect applications with databases.

## Project Structure
- `src/` → Contains source code
- `docs/` → Contains project report
  
## Features
- User registration and login
- Add, update, delete tasks
- Task status tracking
- Search tasks (by name and due date)
- Deadline alerts
- Task sorting (by date and priority)
- View overdue tasks

## Tech Stack
- Python
- MySQL
